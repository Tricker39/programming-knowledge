# 天气种类有多少种?

## 水现象(11 种)

液态降水;雨、阵雨、毛毛雨;固态降水;雪、阵雪、霰、米雪、冰粒、冰雹;混合型降水;雨夹雪、阵性雨夹雪

## 地面凝结现象(4 种)

露、霜、雾凇、雨淞

## 程障碍现象(9 种)

雾、轻雾、吹雪、雪暴、烟幕、霾、沙尘暴、扬沙、浮尘

## 电现象(3 种)

雷暴、闪电、极光

## 他现象(7 种)

大风、飑、龙卷、尘卷风、冰针、积雪、结冰
